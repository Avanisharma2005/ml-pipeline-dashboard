# 🚀 ML Pipeline Dashboard (Streamlit)

An interactive end-to-end Machine Learning pipeline dashboard built using Streamlit.

## 🔥 Features
- Problem type selection (Classification / Regression)
- Data upload & PCA visualization
- Exploratory Data Analysis (EDA)
- Data cleaning (missing values + outlier detection)
- Feature selection (Variance, Correlation, Information Gain)
- Model training with K-Fold validation
- Performance metrics
- Hyperparameter tuning (GridSearch)

## 📊 Dataset Used
Banknote Authentication Dataset (UCI)

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
